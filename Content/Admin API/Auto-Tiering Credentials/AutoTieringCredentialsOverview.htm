Admin API Methods for Auto-Tiering Credentials

This group of Admin API methods is for managing Amazon Web Services (AWS) account credentials in connection with the HyperStore auto-tiering feature. There are API methods for storing AWS credentials in the HyperStore system so they may be used for auto-tiering, for retrieving AWS credentials from the system, and for deleting AWS credentials from the system.

The Cloudian Management Console’s Auto-Tiering Credentials functionality, and the auto-tiering credentials elements within the CMC's interface for Configure a Bucket Lifecycle Policy for Object Auto-Tiering or Expiration, are implemented by calling the methods in this section.

For an overview of the HyperStore auto-tiering feature, see Auto-Tiering Feature Overview.

  • DELETE Amazon Tiering Credentials
  • GET Amazon Tiering Credentials
  • GET Amazon Tiering Credentials Source
  • POST Amazon Tiering Credentials
