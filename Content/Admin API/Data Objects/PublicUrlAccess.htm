PublicUrlAccess

[Admin API Object]

The PublicUrlAccess object is returned as the response payload for this Admin API operation:

  • GET Object's Public URL Permissions

The PublicUrlAccess object is required as the request payload for this Admin API operation:

  • POST Object's Public URL Permissions

The PublicUrlAccess object consists of the attributes below. Each attribute description indicates whether the attribute is mandatory or optional when this object is transmitted as a request payload.

allowRead

( Optional, boolean) Whether a public URL is enabled for the associated object, true or false. Defaults to true.

 currentDownloads

(Optional, number) Current total number of times that the object has been downloaded via public URL. This value is set by the system, not by the client. Starts at 0 for a new public URL.

 expiryTime

(Mandatory, string) Expiration date-time of the public URL in UTC milliseconds. After this date-time the public URL will no longer work.

 maxDownloadNum

(Optional, number) Maximum number of times that the system will allow the object to be downloaded via public URL, by all users in total. To allow unlimited downloads, set this to "-1". Defaults to 1000.

 secure

(Optional, boolean) Whether the object’s public URL should use HTTPS rather than HTTP, true or false. Defaults to true.

 url

(Optional, string) System-generated public URL for accessing the object.

To request that a new public URL be generated for an object, do not include the "url" attribute in the request body.

To modify usage attributes for an existing public URL that the system has already generated for an object (such as to change the "maxDownloadNum" value), use the "url" attribute in the request body to specify the existing public URL.

The public URL for an object will have the following format:

http[s]://<bucketName>.<S3Domain>/<objectName>?AWSAccesKeyId=<accessKeyOfObjectOwner>
&Expires=<expiryTime>&Signature=<signatureString>&x-amz-pt=<internalTrackingCode>

This format follows the AWS specification for signed URLs.

Example

This first example shows a PublicUrlAccess object in the body of a POST request that is requesting that a public URL be created. Note that the "url" attribute is not included in the object.

{"expiryTime":"1444306080000","allowRead":true,"maxDownloadNum":100000,"secure":true}

This is second example shows a PublicUrlAccess object as it is returned in response to a GET request. The object now includes the "url" attribute.

{"expiryTime":"1444306080000","allowRead":true,"currentDownloads":0,"maxDownloadNum":100000,
"secure":true,"url":"https://pubs.bucket.1.s3-region1.mycloudianhyperstore.com/HyperStore-Software-
Install-Guide.html?AWSAccessKeyId=35a39816c26d3e930a70&Expires=1445005200&
Signature=D5dzQ84CIGQvaMQT2hQGF2%2BM0bg%3D&x-amz-pt=MDAyNjgxMjMxNDQ1MDAxNjgwNDIx"}
