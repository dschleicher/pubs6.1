SecurityInfo

[Admin API Object]

The SecurityInfo object is returned as the response payload for these Admin API operations:

  • GET User's S3 Credential
  • GET User's S3 Credentials List
  • GET User's S3 Credentials List (Active)

The SecurityInfo object consists of the following attributes:

accessKey

(string) User’s access key for the HyperStore S3 service.

 active

(boolean) Whether the credential is active, true or false.

 createDate

(string) Creation timestamp for the credential in UTC milliseconds.

 expireDate

(string) Credential expiration date. Currently always null.

 secretKey

(string) User’s secret key for the HyperStore S3 service.

Example

This is an example of a SecurityInfo object.

{"active":true,"secretKey":"JitieOhfVdQ1sQ88yeGQyyLBSQr+aZbU0WreSs12","accessKey":
