GET Group

[Admin API Method]

Use this Admin API method to retrieve the profile of a single specified user group. The profile includes the group’s name and service status (active or inactive).

Request Format

Syntax

GET /group?groupId=xxx

Query Parameters

groupId

(Mandatory, string) Group ID for group whose profile is being retrieved.

Request Payload

None.

 Response Format

Response Status

In addition to the Common Status Codes, this method may return these codes:

Status Code Description
204 Group does not exist
400 Missing required parameters : {groupId}

Response Payload

GroupInfo JSON object.

 Example

In this example a user group’s profile is successfully retrieved. Note that unlike individual user profiles, a group profile has very little information in it.

# Request ->

GET http://192.168.2.16:18081/group?groupId=Dev

# Response ->

Status Code: 200 OK
Content-Length: 59
Content-Type: application/json
Date: Tue, 22 Sep 2015 16:16:35 GMT
{"active":"true","groupName":"Development","groupId":"Dev","ldapGroup":null}

For more information about GroupInfo object attributes, see GroupInfo.
