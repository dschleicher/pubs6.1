DELETE Rating Plan

[Admin API Method]

Use this Admin API method to delete a rating plan.

If users were on the rating plan that you delete, they are automatically switched to their group default rating plan, or to the system default rating plan ("Default-RP") if no group default plan has been set.

You cannot delete the system default rating plan ("Default-RP") or the whitelist rating plan ("Whitelist-RP").

Request Format

Syntax

DELETE /ratingPlan?ratingPlanId=xxx

Query Parameters

ratingPlanId

(Mandatory, string) Unique ID of the rating plan to delete.

Request Payload

None.

 Response Format

Response Status

In addition to the Common Status Codes, this method may return these codes:

Status Code Description
400 Rating Plan does not exist
400 Missing required parameter : {ratingPlanId}

Response Payload

None.

 Example

In this example a rating plan is successfully deleted.

# Request ->

DELETE http://192.168.2.16:18081/ratingPlan?ratingPlanId=Silver

# Response ->

Status Code: 200 OK
Content-Length: 0
Date: Wed, 30 Sep 2015 12:41:07 GMT
