GET Rating Plan

[Admin API Method]

Use this Admin API method to retrieve the contents of a specified rating plan.

Request Format

Syntax

GET /ratingPlan?ratingPlanId=xxx

Query Parameters

ratingPlanId

(Mandatory, string) Unique ID of the rating plan to retrieve.

Request Payload

None.

 Response Format

Response Status

In addition to the Common Status Codes, this method may return these codes:

Status Code Description
204 Rating Plan does not exist
400 Missing required parameter: {ratingPlanId}

Response Payload

RatingPlan JSON object.

 Example

In this example a rating plan is successfully retrieved.

# Request ->

GET http://192.168.2.16:18081/ratingPlan?ratingPlanId=Gold
