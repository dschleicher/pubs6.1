GET Rating Plan List

[Admin API Method]

Use this Admin API method to retrieve a list of all rating plans in the system, identified by ID and name.

The HyperStore system comes with two pre-packaged rating plans. The first is the system default rating plan, for which the ID is "Default-RP". The second pre-packaged rating plan, with ID "Whitelist-RP", supports the billing whitelist feature.

Request Format

Syntax

GET /ratingPlan/list

Query Parameters

None.

Request Payload

None.

 Response Format

Response Status

This method returns only the Common Status Codes.

Response Payload

List<Map<string,string,string,string>>. Strings are: {id,name,encodedid,currency}. "encodedId" value is a URL-encoding of the "id" value.

 Example

In this example the list of rating plans in the system is successfully retrieved.

# Request ->

GET http://192.168.2.16:18081/ratingPlan/list

# Response ->

Status Code: 200 OK
Content-Length: 264
Content-Type: application/json
Date: Wed, 30 Sep 2015 11:40:02 GMT
[{"id":"Default-RP","name":"Default Rating Plan","encodedId":"Default-RP","currency":"USD"},
{"id":"Gold","name":"Gold Level Plan","encodedId":"Gold","currency":"USD"},{"id":"Whitelist-RP",
"name":"Whitelist Rating Plan","encodedId":"Whitelist-RP","currency":"USD"}]
