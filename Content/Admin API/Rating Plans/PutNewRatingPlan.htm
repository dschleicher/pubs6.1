PUT New Rating Plan

[Admin API Method]

Use this Admin API method to create a new rating plan.

Request Format

Syntax

PUT /ratingPlan

Query Parameters

None.

Request Payload

RatingPlan JSON object

 Response Format

Response Status

In addition to the Common Status Codes, this method may return these codes:

Status Code Description
400 Missing required parameter : {id, name}
400 Invalid JSON Object
409 Unique Constraint Violation : {id}

Response Payload

None.

 Example

In this example a new rating plan is successfully created.
