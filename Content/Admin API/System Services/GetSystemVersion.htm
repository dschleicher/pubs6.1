GET System Version

[Admin API Method]

Use this Admin API method to retrieve version information for your HyperStore system.

Request Format

Syntax

GET /system/version

Query Parameters

None.

Request Payload

None.

 Response Format

Response Status

This method returns only the Common Status Codes.

Response Payload

String "text/plain", indicating HyperStore software version and compile timestamp.

 Example

In this example HyperStore version information is successfully retrieved.

# Request ->

GET http://192.168.2.16:18081/system/version

# Response ->

Status Code: 200 OK
Content-Length: 30
Content-Type: text/plain
Date: Wed, 23 Sep 2015 19:46:05 GMT
5.2 Compiled: 2015-08-21 17:21
