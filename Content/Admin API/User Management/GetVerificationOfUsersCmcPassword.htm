GET Verification of User's CMC Password

[Admin API Method]

Use this Admin API method to verify that a user’s supplied password for the Cloudian Management Console (CMC) is the correct password for that user.

Request Format

Syntax

GET /user/password/verify?userId=xxx&groupId=xxx&password=xxx

Query Parameters

userId

(Mandatory, string) User ID for user whose password is being verified.

 groupId

(Mandatory, string) Group ID for user whose password is being verified.

 password

(Mandatory, string) User’s supplied password.

Request Payload

None.

 Response Format

Response Status

In addition to the Common Status Codes, this method may return these codes:

Status Code Description
400 Missing Required parameters : {userId, groupId, password}

Response Payload

String ("true" or "false").

 Example

In this example a user’s CMC password is successfully verified.

# Request ->

GET http://192.168.2.16:18081/user/password/verify?userId=kwring&groupId=Dev&password=nameofmydog

# Response ->

Status Code: 200 OK
Content-Length: 4
Content-Type: text/plain
Date: Wed, 23 Sep 2015 14:44:00 GMT
true
