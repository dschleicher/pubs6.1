GET Whitelist List

[Admin API Method]

Use this Admin API method to retrieve a list of all whitelist IDs and corresponding display names.

This method does not return whitelist content, only the whitelist IDs and names. To retrieve a complete whitelist use the GET Whitelist method.

In the current HyperStore release only one whitelist is supported and its ID is "Default-WL".

Request Format

Syntax

GET /whitelist/list

Query Parameters

None.

Request Payload

None.

 Response Format

Response Status

This method returns only the Common Status Codes.

Status Code Description
xxx xxx
xxx xxx

Response Payload

List<Map<string,string,string>>. Strings are: {id,name,encodedId}. "encodedId" value is a URL-encoding of the "id" value.

 Example

In this example the system’s list of whitelists is successfully retrieved. The HyperStore system does not currently support having multiple whitelists, so this method returns just the one supported list.

# Request ->

GET http://192.168.2.16:18081/whitelist/list

# Response ->

Status Code: 200 OK
Content-Length: 73
Content-Type: application/json
Date: Wed, 30 Sep 2015 13:16:27 GMT
[{"id":"Default-WL","name":"Default Whitelist","encodedId":"Default-WL"}]
