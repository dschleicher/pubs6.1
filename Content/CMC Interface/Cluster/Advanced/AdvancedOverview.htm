Advanced

[CMC Interface]

Path: Cluster → Nodes → Advanced

Supported tasks:

  • Info Commands (hsstool)
    • ring — View vNode info for whole cluster
    • info — View vNode and data load info for a physical node
    • status — View summary status for whole cluster
    • opstatus — View status of repair or cleanup operations
    • hintsstatus — View status of hinted handoff queues
    • proactiverepairq — View status of proactive repair queues
    • repairqueue — View auto-repair schedule
    • ls — View vNode and data load info per mount point
    • whereis — View storage location information for an S3 object
    • metadata — View metadata for an S3 object
  • Maintenance Commands (hsstool)
    • cleanup — Clean a node of replicated data that doesn’t belong to it
    • cleanupec — Clean a node of erasure coded data that doesn’t belong to it
    • clearsnapshot — Delete old backup snapshots of a node
    • decommission — Decommission a live node
    • snapshot — Take a backup snapshot of a node
    • removenode — Remove a dead node from a cluster
    • autorepair — Enable or disable auto-repair
    • repair — Repair the replicated data on a node
    • repairec — Repair the erasure coded data on a node
  • Redis Monitor Operations
    • setClusterMaster — Move the Redis Credentials or Redis QoS master role
    • getClusterInfo — Get information about a Redis Credentials or Redis QoS cluster
  • Disk Management
    • disableDisk — Temporarily disable a HyperStore data disk
    • enableDisk — Re-enable a disabled HyperStore data disk
    • replaceDisk — Activate a replacement HyperStore data disk and restore data to it
  • Collect Diagnostics
  • Add/Uninstall Node
    • Add a Node
    • Removing a Node
