View Status of Nodes Per Data Center

[CMC Interface]

The CMC's Data Centers page displays a panel for each data center in your HyperStore system. For each data center, each HyperStore node in the data center is represented by a cube. The cube color is either:

  • Orange — Indicates that the node has alerts that have not yet been acknowledged by a system administrator. A failed disk on the node will also cause the node to display as orange.
  • Green — Indicates that the node has no unacknowledged alerts.

To see a node’s hostname and summary statistics, hold your cursor over a cube.

The display shows the same summary status information as is available on the Node Status page. For description of the status items, see View a Node's Summary Status.

Drill Down to a Node’s Detailed Status

Click on a node’s cube icon to open the CMC Node Status page for that node. On that page you can view summary and detailed status information for the node, start or stop services on the node, or review and acknowledge node alerts.
