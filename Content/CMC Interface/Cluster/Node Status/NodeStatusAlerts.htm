View and Acknowledge Node Alerts

[CMC Interface]

At the bottom of the CMC's Node Status page is an Alert List section that displays a list of node alerts (for the one node that you’ve selected at the top of the page). In this section you can review and acknowledge node alerts.

This section has the same functionality as the CMC’s Alerts page, except that the information is limited to the one node that you’ve selected. For more guidance on understanding and working with this display, see Alerts.
