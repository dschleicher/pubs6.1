Node Status

[CMC Interface]

Path: Cluster → Nodes → Node Status

Supported tasks:

  • View a Node's Summary Status
  • View a Node's Disk Status
  • View a Node's Memory Status
  • View a Node's Services Status
  • Start, Stop, or Restart Services On a Node
  • View and Acknowledge Node Alerts

If your HyperStore system has multiple service regions, a drop-down list displays at the top of the page so you can select a region first before selecting a node for which to view status.
