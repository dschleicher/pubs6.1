Cloudian Management Console (CMC)

The Cloudian Management Console (CMC) is a web-based user interface for Cloudian HyperStore system administrators, group administrators, and end users. The functionality available through the CMC depends on the user type associated with a user’s login ID (system admin, group admin, or regular user). System admins can perform a wide range of system maintenance and operational tasks as well as provisioning and managing user accounts. Group admins can perform a much more limited range of tasks, pertaining specifically to their group. Regular users can use the CMC to create and configure storage buckets and to upload or download data.

Just as the CMC's functionality is tailored to the logged-in user type, so too is the CMC's online help: users can only view the Help topics for the functionality that is available to them based on their user type.

The Cloudian Management Console (CMC) is a web-based user interface to the Cloudian HyperStore object storage system. As a group administrator you can use this interface to perform management tasks for your group, such as adding users or generating service usage reports. You can also create and configure storage buckets and upload or download data.

The Cloudian Management Console (CMC) is a web-based user interface to the Cloudian HyperStore object storage system. As an end user of the storage system, you can use this interface to create and configure storage buckets and to upload or download data. You can also use it for account management tasks such as changing your password.

The CMC has been tested most extensively with the Firefox browser. It is recommended that you use a recent version of Firefox when accessing the Console. Internet Explorer is not supported — you may experience display problems with some CMC pages if you use IE. There are no known problems with using Chrome to access the CMC, but Chrome support has not been thoroughly tested.
