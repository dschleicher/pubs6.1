Security Credentials

[CMC Interface]

Path: Drop-down menu under your user name (at top right) → Security Credentials

Supported task:

  • Change your CMC password

The Security Credentials page also supports a function for managing S3 access credentials. However, this function does not display for system admins since the system admin role does not have its own S3 storage service account. To learn more about this function you can log into the CMC as a group admin or regular user and check the online Help.

To change your CMC password:

  1. In the "New Password" field, enter a new password that you will use to log into the CMC. Maximum supported password length is 64 characters.
  2. In the "Confirm Password" field, enter the new password again.
  3. Click Change Password.
