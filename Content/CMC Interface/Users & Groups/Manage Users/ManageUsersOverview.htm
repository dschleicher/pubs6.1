Manage Users

[CMC Interface]

Path: Users & Groups → Manage Users

Supported tasks:

  • Add a User
  • Work with existing users:
    • Retrieve a User or List of Users
    • Edit a User
    • Set Quality of Service for Users and Groups
    • Manage a User's Stored Objects
    • Delete a User

For an overview of provisioning HyperStore service users, see User Provisioning Feature Overview.
