Operating System Requirements

Cloudian HyperStore 6.0 is supported only on these versions of Linux:

  • Red Hat Enterprise Linux (RHEL) 6.x, 64-bit
  • CentOS 6.x, 64-bit

Cloudian recommends using version 6.7 or a later 6.x version.

If you try to install HyperStore software on a host that is not RHEL or CentOS 6.x, or is not 64-bit, the installation will abort.
