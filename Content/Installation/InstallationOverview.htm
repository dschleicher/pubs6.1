HyperStore Installation

This documentation is for Cloudian HyperStoreR version 6.0. For more information about the 6.0 release, see What's New.

The main part of this documentation describes how to do a fresh installation of Cloudian HyperStore 6.0.

If you are upgrading to 6.0 from version 5.2 or later, you can go directly to Upgrading to HyperStore 6.0 From Version 5.2 or Later. If you want to upgrade to 6.0 from a version older than 5.2, please contact Cloudian Support for instructions specific to your particular upgrade path.

If you do not yet have the HyperStore 6.0 package, you can obtain it from the Cloudian FTP site ftp.cloudian.com. You will need a login ID and password (available from support@cloudian.com). Once logged into the FTP site, change into the Cloudian_HyperStore directory and then into the cloudian-6.0 sub-directory. From here you can download either:

* The 6.0 HyperStore appliance ISO file CloudianHyperStore-6.0-CentOS-6.7-x86_64-Install.iso.
* The 6.0 HyperStore software-only release package CloudianHyperStore-6.0.bin.

Beneath the cloudian-6.0 sub-directory, product documentation is available under the /doc sub-directory — either as a set of HTML documents, or as a tarball or zip package of those same documents. The same documents are also bundled inside the ISO download and inside the software-only package download.

To install and run HyperStore software you need a HyperStore license file — either an evaluation license or a production license. If you do not have a license file you can obtain one through Cloudian Support (support@cloudian.com) or from your Cloudian sales representative. If you are using the appliance ISO, the ISO package includes a very low-capacity license that can be used temporarily for verifying the functionality of your HyperStore system after installation, if you haven’t yet obtained an evaluation or production license.
