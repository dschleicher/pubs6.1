Checking Disk Status

You can use the CMC to check on the current status of any disk in your HyperStore system. Go to Cluster → Nodes → Node Status, and select a node to check on. Then review the Disk Detail Info section to see the current status of each disk on the selected node. Each disk’s status is communicated by a color-coded icon, with the status being one of OK, Error, Disabled, or Disabled - No Move.
