S3 Storage Interface Feature Overview

Broadly you have two options for using the S3 API to put objects into the HyperStore object store, retrieve objects, and so on:

  • Use the Cloudian Management Console (CMC). The CMC has a built-in S3 client that allows you — and your users, if you give them access to the CMC — to interact with the HyperStore object store through a graphical interface.
  • Use a third party S3 client or an S3 client that your organization develops, to directly invoke HyperStore’s implementation of the S3 API.
