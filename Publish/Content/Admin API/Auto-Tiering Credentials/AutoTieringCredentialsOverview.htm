Admin API Methods for Auto-Tiering Credentials

This group of Admin API methods is for managing Amazon Web Services (AWS) account credentials in connection with the HyperStore auto-tiering feature. There are API methods for storing AWS credentials in the HyperStore system so they may be used for auto-tiering, for retrieving AWS credentials from the system, and for deleting AWS credentials from the system.

Note  For an overview of the HyperStore auto-tiering feature, see Auto-Tiering Feature Overview.
