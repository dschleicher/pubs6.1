Bill

[Admin API Object]

The Bill object is returned as the response payload for these Admin API operations:

  • POST User's Bill
  • GET User's Bill

The Bill object consists of the following attributes:

billID

(string) Unique bill ID.

canonicalUserId

(string) System-generated canonical user ID for the user.

currency

(string) Currency string.

endCal

(string) End date/time of the billing period in UTC milliseconds.

groupId

(string) ID of the group to which the user belongs.

notes

(string) Notes regarding the bill, if any.

regionBills

(Map<string,RegionBill>) List of bills by region. In a single-region HyperStore deployment there will be only one RegionBill.

startCal

(string) Start date/time of the billing period in UTC milliseconds.

total

(number) The total charges incurred by the user during the billing period, excluding activity for whitelisted source IP addresses.

userId

(string)  ID of the user for whom the bill was generated.

whitelistTotal

(number)  The total charges incurred by the user for activity originating from whitelisted source IP addresses.
