Locate a Disk on a HyperStore 1500 or 3000 Appliance

[CMC Interface]

The Appliance section of the CMC's Node Status page is available only if you are using an HSA1500 or FL3000 series HyperStore appliance. In the event of a disk failure (as indicated by an alert message and by the Disk Detail Info section of the Node Status page), you can open the Appliance panel for information and tools to help you quickly locate the failed disk within your physical HyperStore environment.

The screen image below is for the FL3000.

For the FL3000, if the failed disk is an SSD (storing the OS) and you need assistance identifying the physical micronode on which the SSD resides, you can click the Blink Node button and the micronode’s ID light will blink for three minutes. If the failed disk is an HDD (storing HyperStore data) and you need assistance identifying the JBOD enclosure in which the HDD resides, click Blink JBOD and the JBOD enclosure’s ID light will blink for three minutes. See the diagrams for the location of the ID lights.

For the HSA1500, if you need assistance identifying the physical node on which the failed disk resides, click Blink Node and the node’s ID light will blink for three minutes.

For either type of appliance, the disk table in the lower portion of the display provides buttons for blinking the ID light of the particular disk that has failed. The disk table itself does not indicate disk status — you can get the failed device name from the Disk Detail Info section of the Node Status page, then locate that device within the disk table in the Appliance section and click that disk’s Blink button. The disk’s ID light will then blink for three minutes.

The disk table also indicates each disk’s serial number and slot number. In the event that the blink light is not working, you can use the slot information and the diagrams to physically locate the failed disk.

The Appliance section also has links to the hardware manuals for your appliance (PDF files stored on your CMC nodes).

Note  To enlarge an appliance diagram in your browser, you can press <Ctrl>+ (and then <Ctrl>- to shrink the display back to regular size).
