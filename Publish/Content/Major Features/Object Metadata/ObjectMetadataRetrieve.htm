Retrieving Object Metadata

S3 client applications can retrieve object metadata by using either of two standard Amazon S3 API methods, both of which the HyperStore system supports:

  • GET Object returns system-defined and user-defined object metadata (as response headers) as well as returning the object itself.
  • HEAD Object returns system-defined and user-defined object metadata (as response headers) as well as returning the object itself.

Additionally, HyperStore support for the GET Bucket (List Objects) method includes an extension that enables S3 clients to retrieve user-defined metadata (as well as system-defined metadata) for all of the objects in a specified bucket.
