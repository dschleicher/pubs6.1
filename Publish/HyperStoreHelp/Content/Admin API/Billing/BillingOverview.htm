Admin API Methods for Billing

This group of Admin API methods is for generating a billable activity report for a specified user, or retrieving a billable activity report that has already been generated. The report shows the user’s billable activity for a specified calendar month, and the charges for that activity based on the user’s assigned rating plan.

Note  The Cloudian Management Console’s Account Activity functionality is implemented by calling the methods in this section.

Note  For an overview of the HyperStore billing feature, see Billing Feature Overview.

IMPORTANT: If you want to bill for data upload or download volume, or for HTTP request volume, you must enable the Track/Report Usage for Request Rates and Data Transfer Rates setting in the CMC’s Configuration Settings page. By default this setting is disabled and the system does not maintain per-user HTTP request counts and data transfer byte counts.
