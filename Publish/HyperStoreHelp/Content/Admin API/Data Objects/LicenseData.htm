LicenseData

[Admin API Object]

The LicenseData object is returned as the response payload for this Admin API operation:

  • GET License Info

The LicenseData object consists of the following attributes:

expiration

(number) License expiration in UTC milliseconds.

gracePeriod

(number) After license expiration, the number of days until the HyperStore service is automatically disabled.

maxNetStorage

(number) Total allowed stored object size for the entire HyperStore deployment, across regions, in bytes. This limit does not account for replication. For example, if your deployment is currently storing S3 objects totalling 100GB in size, and your replication factor is 3, then this storage counts as 100GB toward the "maxNetStorage" limit — not as 300GB.

A value of -1 indicates unlimited storage.

warnPeriod

(number)  Starting this many days before the license expiration date, an expiration warning message will display at the top of the Cloudian Management Console.

Example

This is an example of a LicenseData object.

{"expiration":1718311716965,"warnPeriod":30,"gracePeriod":0,"maxNetStorage":104857600}
