Add a Group

[CMC Interface]

  1. In the CMC's Manage Groups page, click New Group. This opens the Add New Group panel.

  1. In the Add New Group panel, complete the group information:
Group Name
  • Must be unique within your entire HyperStore service.
  • Only letters, numbers, dashes, and underscores are allowed.
  • Maximum allowed length is 64 characters.
LDAP Org Unit
  • This field displays only if LDAP Authentication is enabled. For information about the LDAP Authentication feature see Integrating with an LDAP System.
  • Use this field to specify the "ou" (Organizational Unit) value for this group from your LDAP system. Make sure the value you enter here exactly matches the group’s LDAP "ou" value.
  • This field does not have the character restrictions that the "Group Name" field has. For example, if the group’s "ou" value in LDAP is "Quality Assurance (U.S.)" you would enter that value exactly in the "LDAP Org Unit" field, but in the "Group Name" field you would need to enter something like "Quality_Assurance_US". The CMC will use the "LDAP Org Unit" value for purposes of LDAP authentication, while using the "Group Name" value internally for adherence to HyperStore data schemas. Users within the group will use the "Group Name" value when selecting from the Group Name drop-down list when logging into the CMC.
Group Description
  • Maximum allowed length is 64 characters.
Rating Plan
  • Rating plan to assign to the group for billing calculation purposes. This rating plan will apply to each user in the group, with the exception of any users to whom you individually assign a different rating plan.
  • If you don’t choose a rating plan for the group, the Default Rating Plan (Default-RP) is automatically assigned to the group.
Note for multi-region systems

If your service deployment has multiple service regions, rating plan assignment is on a per-region basis. A "Region" drop-down list displays which includes each of your region names. For each region, assign the group a rating plan that will apply to the group’s service use within that region.

  1. Click Save.
