Appliance Set-Up

If you have a bare metal HyperStore appliance machine and the HyperStore appliance ISO file ( CloudianHyperStore-6.0-CentOS-6.7-x86_64-Install.iso), set up the machine by following the instructions in Installing CentOS from the HyperStore ISO File. This not only installs the OS but also enables you to set up the host's file system and basic networking settings through a wizard-type interface.
