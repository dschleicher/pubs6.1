Cloudian Management Console (CMC)

The Cloudian Management Console (CMC) is a web-based user interface to the Cloudian HyperStore object storage system. As a group administrator you can use this interface to perform management tasks for your group, such as adding users or generating service usage reports. You can also create and configure storage buckets and upload or download data.

Note  The CMC has been tested most extensively with the Firefox browser. It is recommended that you use a recent version of Firefox when accessing the Console. Internet Explorer is not supported — you may experience display problems with some CMC pages if you use IE. There are no known problems with using Chrome to access the CMC, but Chrome support has not been thoroughly tested.
