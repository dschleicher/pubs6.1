Add a User

To add a user:

  1. In the CMC's Manage Users page, click New User. This opens the Add New User panel.

Note for systems using LDAP authentication

If system administrators have configured the storage system to use LDAP authentication, the system automatically provisions users when they log into the CMC with their LDAP credentials. Do not use the CMC’s Add New User function if the system is configured to use LDAP authentication. Users that you add through the CMC’s Add New User function will not be able to log in if the CMC is using LDAP authentication.

  1. In the Add New User panel, complete the following user information:
User ID
  • Must be unique within the group.
  • Only letters, numbers, dashes, and underscores are allowed.
  • Maximum allowed length is 64 characters.
  • The following user IDs are reserved for system use and are not available to individual users: "anonymous", "public", "null", "none", "admin", "0".
User Type

From the drop-down list select the type of user that you want to create:

  • User -- A regular user of the storage service. He or she will be able to use the CMC to create storage buckets, upload and download objects, display reports on their service usage, and manage their service access credentials.
  • Group Admin -- An administrator of your group. He or she will be able to use the CMC to perform administrative functions for the group, such as adding users or setting user-level QoS profiles. A group admin has a storage service account and can upload their own objects to storage. A group admin can also manage stored objects on behalf of particular users within the group.
Group Name

This field displays your group name and cannot be edited.

Password / Confirm Password

The user's password for logging into the CMC. Maximum allowed length is 64 characters.

  1. Optionally, click More to display additional fields:

Full Name (maximum 64 characters)

Address and contact information

  1. Click Save.
