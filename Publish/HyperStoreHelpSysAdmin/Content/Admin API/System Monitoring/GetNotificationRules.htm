GET Notification Rules

[Admin API Method]

Use this Admin API method to retrieve the list of notification rules that currently exist in the HyperStore system. This includes the set of rules that are included in the system by default and also any rules that you have created.

Request Format

Syntax

GET /monitor/notificationrules[?region=xxx]

Query Parameters

region

(Optional, string) Service region from which to retrieve the notification rule list. In a multi-region system each region has its own notification rule list. If you do not specify a region, the default regionâ€™s rule list will be retrieved.

Request Payload

None.

Response Format

Response Status

This method returns only the Common Response Status Codes

Response Payload

List <NotificationRule> in JSON.

Example
