Admin API Methods for User Management

This group of Admin API methods is for managing HyperStore service user accounts. This includes support for creating, editing, and deleting accounts. It also supports management of users' security credentials and the assignment of rating plans to users.

Note  The Cloudian Management Console’s Manage Users functionality is implemented by calling the methods in this section.

Note  For an overview of HyperStore user provisioning, see User Provisioning Feature Overview.

Note  This group of methods does not support the assignment of quality of service settings to user accounts — for that, see the POST QoS Limits method.
