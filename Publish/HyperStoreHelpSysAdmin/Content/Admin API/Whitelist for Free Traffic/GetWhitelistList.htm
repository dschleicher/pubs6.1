GET Whitelist List

[Admin API Method]

Use this Admin API method to retrieve a list of all whitelist IDs and corresponding display names.

This method does not return whitelist content, only the whitelist IDs and names. To retrieve a complete whitelist use the GET Whitelist method.

Note  In the current HyperStore release only one whitelist is supported and its ID is "Default-WL".

Request Format

Syntax

GET /whitelist/list

Query Parameters

None.

Request Payload

None.

Response Format

Response Status

This method returns only the Common Response Status Codes

Status Code Description
xxx xxx
xxx xxx

Response Payload

List<Map<string,string,string>>. Strings are: {id,name,encodedId}. "encodedId" value is a URL-encoding of the "id" value.
