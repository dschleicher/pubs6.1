Delete a User

[CMC Interface]

You can delete a HyperStore service user from the system. If you delete a user, their data is deleted from storage. If you want to temporarily deny service access to a user without deleting their stored objects, donâ€™t delete the user. Instead, suspend the user.

To delete a user:

  1. Use the CMC's Manage Users page to Retrieve a User or List of Users.
  2. In the "Actions" column for the user that you want to delete, click Delete.
  3. When prompted by the system, confirm that you want to delete the user.
